# Discover Meteor

In order to run Meteor locally follow these instructions below:

### Install Meteor

```html
curl https://install.meteor.com | sh
```

### Creating a Simple App

```html
meteor create microscope
```

### Adding a Package

```html
meteor add package_name
```

### Run Meteor Locally

```html
meteor
```
